# 💤 LazyVim

```
Phím tắt Vim cơ bản
Dưới đây là các phím tắt Vim quan trọng:
Chế độ (Modes)

Esc - Chuyển sang Normal mode
i - Chuyển sang Insert mode (chèn văn bản trước con trỏ)
a - Chuyển sang Insert mode (chèn văn bản sau con trỏ)
v - Chuyển sang Visual mode (chọn văn bản)
: - Chuyển sang Command mode

Di chuyển trong Normal mode

h - Di chuyển sang trái
j - Di chuyển xuống dưới
k - Di chuyển lên trên
l - Di chuyển sang phải
w - Di chuyển đến đầu từ tiếp theo
b - Di chuyển đến đầu từ trước đó
0 - Di chuyển đến đầu dòng
$ - Di chuyển đến cuối dòng
gg - Di chuyển đến đầu file
G - Di chuyển đến cuối file
:{số} - Di chuyển đến dòng có số chỉ định

Sửa văn bản

x - Xóa ký tự dưới con trỏ
dd - Xóa dòng hiện tại
yy - Sao chép dòng hiện tại
p - Dán sau con trỏ
P - Dán trước con trỏ
u - Hoàn tác thay đổi
Ctrl + r - Làm lại thay đổi đã hoàn tác

Tìm kiếm và thay thế

/từ_khóa - Tìm kiếm từ khóa
n - Đi đến kết quả tìm kiếm tiếp theo
N - Đi đến kết quả tìm kiếm trước đó
:%s/cũ/mới/g - Thay thế tất cả "cũ" bằng "mới"

Lưu và thoát

:w - Lưu file
:q - Thoát (nếu không có thay đổi)
:wq hoặc :x - Lưu và thoát
:q! - Thoát không lưu thay đổi
```
